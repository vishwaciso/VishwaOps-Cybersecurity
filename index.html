Common Prerequisites

1. Networking Fundamentals

Work: Configure firewalls, troubleshoot connectivity, understand TCP/IP, DNS, VPNs

Tools: Wireshark, Nmap, Cisco Packet Tracer

Certs: CompTIA Network+, Cisco CCNA

2. Operating Systems

Work: Administer Windows & Linux, manage logs, users, permissions

Tools: Event Viewer, Sysmon, journalctl

Certs: CompTIA Linux+, Microsoft MTA/MCSA

3. Programming / Scripting

Work: Automate tasks, write security scripts, analyze malware

Tools: Python, Git, VSCode

Certs: Python Institute, OSWE

3a. Shell Scripting (Bash)

Work: Automate Linux admin, logs, cronjobs

Tools: Bash/Zsh, sed, awk, grep

Certs: LPIC-1, Linux+

How: Write scripts to detect anomalies, automate patching

3b. PowerShell Scripting

Work: Automate Windows sec, manage AD, logs

Tools: PowerShell ISE, Empire, BloodHound

Certs: Microsoft MCSA, SANS SEC505

How: Scripts for user mgmt, log scraping, defense

4. Cybersecurity Fundamentals

Work: CIA triad, threats, encryption

Tools: Burp Suite, Nessus, Hashcat

Certs: Security+, SSCP

5. Basic Security Tools

Work: IDS/IPS, AV, logs

Tools: Snort, Suricata, Splunk, Zeek

Certs: Splunk Core, Cisco CyberOps
