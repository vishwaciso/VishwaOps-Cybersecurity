CYBERSECURITY LEARNING PATH

Your roadmap to becoming a cybersecurity professional

Networking Operating Systems Programming Cyber Fundamentals Security Tools Cloud Security App Security
Common Prerequisites

1. Networking Fundamentals

Core Concepts

  • TCP/IP model and OSI layers
  • IP addressing and subnetting
  • DNS, DHCP, ARP protocols
  • Routing and switching fundamentals
  • Network topologies and architectures

Security Aspects

  • Firewall configuration (ACLs, rulesets)
  • VPN technologies (IPSec, SSL/TLS)
  • Network segmentation strategies
  • Intrusion Detection/Prevention Systems
  • Packet analysis and traffic monitoring

Key Tools

Wireshark: Packet capture and analysis

Nmap: Network discovery and security auditing

Cisco Packet Tracer: Network simulation

tcpdump: Command-line packet analyzer

Postman: API testing tool

Certifications

CompTIA Network+: Vendor-neutral networking

Cisco CCNA: Cisco-specific networking

JNCIA-Junos: Juniper networks

WCNA: Wireshark certified

2. Operating Systems

Windows Administration

  • Active Directory management
  • Group Policy implementation
  • User and permission management
  • Windows Registry fundamentals
  • PowerShell scripting
  • Windows Server administration
  • Event log analysis

Linux Administration

  • Filesystem hierarchy standard
  • User/group permissions (chmod, chown)
  • Package management (apt, yum)
  • Service management (systemd)
  • Bash scripting
  • Kernel tuning and modules
  • Log analysis (journalctl, syslog)

Common Tools

Windows: Event Viewer, Sysmon, Process Monitor

Linux: top/htop, ps, lsof, netstat, grep

Cross-platform: RSAT, WinSCP, PuTTY

Monitoring: Nagios, Zabbix, Prometheus

Certifications

CompTIA Linux+: Linux administration

Microsoft MTA/MCSA: Windows Server

LPIC: Linux Professional Institute

RHCSA: Red Hat Certified SysAdmin

3. Programming/Scripting

Python for Security

  • Network scanning and packet crafting
  • Web scraping and automation
  • Malware analysis scripts
  • Cryptography implementations
  • API interactions for security tools
  • Data analysis for security logs

Other Languages

C/C++: Low-level programming, exploit development

JavaScript: Web security, XSS, CSRF

SQL: Database security, injection

Go/Rust: Secure system programming

Assembly: Reverse engineering

Development Tools

Git: Version control for security tools

VSCode: Lightweight code editor

Jupyter Notebooks: Data analysis

Docker: Containerized testing

Postman: API testing

Certifications

Python Institute: PCAP, PCPP

OSWE: Advanced Web Exploitation

GIAC GPPA: Python for Pentesters

Core Principles

  • CIA Triad (Confidentiality, Integrity, Availability)
  • Authentication vs Authorization
  • Risk management frameworks
  • Security governance and compliance
  • Security models (Zero Trust, Defense in Depth)

Threat Landscape

  • Common attack vectors and techniques
  • Malware types and analysis
  • Social engineering tactics
  • OWASP Top 10 vulnerabilities
  • MITRE ATT&CK framework

Security Tools

Burp Suite: Web application testing

Nessus/OpenVAS: Vulnerability scanning

Hashcat: Password cracking

Metasploit: Penetration testing

Kali Linux: Security distro

Certifications

CompTIA Security+: Entry-level security

ISC2 SSCP: Systems security

CEH: Ethical hacking

eJPT: Junior pentesting

Protection Systems

  • Firewalls (pfSense, Cisco ASA)

  • Email security gateways

Monitoring & SIEM

  • ELK Stack (Elasticsearch, Logstash, Kibana)

  • Security Onion distro

Network Tools

Certifications
