Cybersecurity Career Learning Path

Common Prerequisites

1. Networking Fundamentals

Work: Configure firewalls, troubleshoot connectivity, understand TCP/IP, DNS, etc.

Tools: Wireshark, Nmap, Cisco Packet Tracer

Certs: CompTIA Network+, Cisco CCNA

2. Operating Systems

Work: Administer Windows & Linux, manage logs, users, permissions, CLI.

Tools: Event Viewer, Sysmon, journalctl, top, ps

Certs: CompTIA Linux+, Microsoft MTA/MCSA

3. Programming/Scripting

Work: Automate tasks, write security scripts, analyze malware, develop tools.

Tools: Python, Git, VSCode, Jupyter Notebooks

Certs: Python Institute, Offensive Security (OSWE)

4. Cybersecurity Fundamentals

Work: Learn CIA triad, analyze threats/vulns, apply encryption/hashing.

Tools: Burp Suite, Nessus, Hashcat, OpenVAS

Certs: CompTIA Security+, ISC2 SSCP

5. Basic Security Tools

Work: Use IDS/IPS, AV, Wireshark, log monitoring.

Tools: Snort, Suricata, Splunk, ELK, Zeek

Certs: Splunk Core Certified, Cisco CyberOps Associate

Cloud Security

Work: Secure AWS/Azure/GCP, IAM, containers

Tools: CSPM, Prisma, Wiz

Certs: CCSK, AWS Sec Spec

Application Security

Work: Secure apps from exploits, code review

Tools: Burp, ZAP, SAST/DAST

Certs: OSWE, CSSLP, CEH

Network Security

Work: Manage firewalls, VPN, IDS/IPS

Tools: Cisco, Palo Alto, Check Point

Certs: CCNA, PCNSE

SOC/CSIRT

Work: Threat detection, incident response

Tools: SIEM, EDR, Splunk

Certs: CySA+, GCIA, GCED

Identity & Access

Work: Manage IAM, MFA, privilege mgmt

Tools: Okta, AD, SailPoint

Certs: CISM, Okta, CISSP

DevSecOps

Work: Secure CI/CD, IaC pipelines

Tools: Jenkins, GitLab, K8s

Certs: CKS, HashiCorp

Application Security

Work: Harden apps, secure frameworks

Tools: RASP, SCA, WAF

Certs: OSWE, CAST, GIAC

Cloud Risk

Work: Cloud audits, compliance

Tools: CSPM, GRC Suites

Certs: CISA, CCSP, ISO27k

Security Awareness

Work: Train staff, build awareness

Tools: LMS, PhishSim

Certs: CompTIA Security+

IoT/OT Security

Work: Secure IoT devices, ICS/SCADA

Tools: Claroty, Nozomi, Dragos

Certs: GICSP, ISA/IEC 62443
